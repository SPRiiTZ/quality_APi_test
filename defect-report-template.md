# Defect Report Template

| Felt | Indhold |
|---|---|
| Defect ID | Fx DEF-001 |
| Titel | Kort fejlbeskrivelse |
| Severity | Critical / High / Medium / Low |
| Priority | High / Medium / Low |
| API endpoint / Modul | Fx `/api/discount` eller `QualityService` |
| Preconditions | Hvad skal være klart |
| Steps to Reproduce | Nummererede trin |
| Expected Result | Forventet resultat |
| Actual Result | Faktisk resultat |
| Test Type | Black box / Unit test / CI |
| Driftstilstand / Konfiguration | Fx production mode, maintenance mode, loyal customer |
| Environment | Browser, OS, dato |
| Reporter | Navn eller gruppe |
| Status | New / Open / Closed |
